Readme

Les principales difficultés rencontrées lors du codage du site sont en lien avec notre adaption au langage javascript. C'est un langage compliqué qui est très différent du python. Ainsi, dans la totalité de nos codes, nous avons rencontrés des difficultés liées à ça.

Réflexes

La difficulté rencontrée dans le codage de ce jeu était d'empêcher que ne bouton soit clickable au mauvais moment

La principale difficulté rencontrée dans ce jeu est le problème d'arrondi.

Simon :

Nous avons rencontrés des problèmes en maniant le langage javascript, qui est difficile et très différent du python. De plus, nous avons eu un problème pour relier le css et le javascript au fichier html pendant un certains temps. La raison principale était une erreur dans le nom de fichier. Cela a posé problème car le css est indispensable pour le jeu: c'est grâce à cela que nous avons les couleurs.

Mots couplés (projet abandonné) :

Nous avions pour projet de coder le jeu des mots couplés, mais nous avons très vite abandonné cette idée. En effet, nous avions préalablement codé ce jeu sur python, et cétait déjà plutôt compliqué. Le projet fais appel à des fichiers extérieurs, (pour vérifier si les mots existent), ce qui complique vraiment la tâche en javascript. Enfin, nous avons préféré opter pour des minis jeux, plus faciles à faire, pour en avoir plusieurs. Nous avons donc remplacé ce jeu par le Jeu de Nim.

Jeu de Nim :

Pour ce jeu, nous avons eu des difficulté pour intégrer une variable dynamique pour afficher en temps réel le nombre de coups et le nombre d'allumettes restantes. Nous avons trouver ${}.

Calcul Mental
